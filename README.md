# Oefenreeks 02 - Herhaling omvormen list naar navigatie
## Oefening C
### OPGAVE/ STAPPEN
* Werk verder op de uitwerking van oefening B als uitgangspunt voor je uitwerking

Voer volgende stappen uit met een editor naar keuze.

* Voorzie op de pagina’s index.html en tabellen.html een passende navigatiebalk.
  * Deze navigatiebalk is opgemaakt door middel van een unordered list element.
  * De navigatiebalk zal voor de pagina waar deze opstaat een afzonderlijke opmaak hebben.
* Experimenteer even met de mogelijkheden om je eigen stijl aan deze navigatiebalk toe te voegen.
  * Ga even online op zoek naar mogelijke CSS properties die je navigatiebalk een persoonlijk touch kunnen geven.
